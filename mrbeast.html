Работа над ошибками. Позиционирование.

Каждый пункт меню <li> получает свойство position: relative;. Вложенные элементы с position: absolute; будут отталкиваться от позиций главного меню. Каждое подменю будет позиционироваться относительно своего родительского пункта, а не всего потока.

Вложенные списки подменю (элементы ul.submenu) имеют position: absolute;, поэтому они появляются сбоку от родителя. Смещения задаются свойствами:

.submenu {
  position: absolute;
  left: 100%;
  top: 0;
}

left: 100% сдвигает подменю вправо ровно на ширину родительского пункта, поэтому оно появляется рядом, а не поверх него.
top: 0 выравнивает верх подменю по верхнему краю родительского элемента.

Каждый пункт подменю также имеет position: relative;, чтобы можно было добавить второй уровень вложенности. Второе подменю наследует тот же принцип:

.submenu .submenu {
  position: absolute;
  left: 100%;
  top: 0;
}

В итоге первое подменю появляется справа от основного пункта, второе подменю появляется справа от первого и т.д.

При наведении на пункт меню включается CSS-правило:

.vmenu li:hover > .submenu,
.submenu li:hover > .submenu {
  display: block;
}

Оно делает вложенный список видимым (display: block) и создаёт эффект раскрывающегося многоуровневого меню.
